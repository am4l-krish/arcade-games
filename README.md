# 🕹️ SideQuest Arcade

A retro-style arcade game website featuring 5 classic mini games built with HTML, CSS and JavaScript.

## 🎮 Games

| Game | Description |
|------|-------------|
| ❌⭕ Tic Tac Toe | Classic 2-player strategy game |
| 🃏 Memory Game | Flip and match cards to win |
| 🐛 Whack a Bug | Hit as many bugs as you can before time runs out |
| 🦆 Running Duck | Dodge obstacles and survive as long as possible |
| 👾 Space Invaders | Shoot down the aliens before they reach you |

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Press Start 2P Font (Google Fonts)

## 🚀 How to Run

1. Clone this repository
2. Add your own media files (see below)
3. Open `index.html` in your browser
4. Select a game and play!

> **Note:** Best experienced on desktop. Some games require keyboard controls.

## 🎬 Media Files

Video and audio files are not included due to file size limits.

To run locally, add these files manually:

| File | Location |
|------|----------|
| `main-bg.mp4` | root folder |
| `game-bg.mp4` | `common/` folder |
| `music.mp3` | root folder |
